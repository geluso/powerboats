Welcome to Powerboats

The Powerboats board game is created by Cwali. This implementation is created by Steve Geluso, a fan. Check out Cwali's website to see more about the actual board game and all their other projects. I highly encourage you to obtain a copy of the board game itself. Unfortunately I believe it is out of publication and you'll need to turn to websites like Ebay or the Amazon marketplace to purchase a used version of it.

Click learn from the nav bar above to learn a bit about how this game works.
